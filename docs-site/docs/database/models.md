# Модели данных

## User

Пользователь системы. Хранит логин, хеш пароля, аватар, роль и дату создания.

## Recipe

Рецепт пользователя. Содержит название, описание, статус, автора, ингредиенты, главное изображение и даты создания и изменения.

## RecipeStep

Шаг приготовления рецепта. Содержит id рецепта, номер шага, текстовое описание и ссылку на изображение.

## CartItem

Элемент записи рецепта в продуктовой корзине. Содержит id рецепта, id пользователя и число порций.

## Favorite

Связь User и Recipe -> "избранное". Содержит id рецепта, id пользователя и дату создания.

## Note

Заметка к рецепту. Содержит id рецепта, id пользователя, название, описание, а также даты создания и редактирования.

## Comment

Комментарий к рецепту или статье. Содержит автора, id рецепта / статьи, текст и дату создания и обновления.

## Article

Статья. Содержит автора (администратор), название, текст, ссылку на изображение, а также даты создания и редактирования.

## Chef

Статья о поваре. Содержит автора, имя повара, описание, ссылку на изображение, теги, а также дату создания.
