# Тестирование функциональности блока КБЖУ в описании рецепта


### Сценарий 1

Видимость блока КБЖУ в описании рецепта.

#### Предусловие

- Рецепт опубликован;
- Автор в описании рецепта заполнил хотя бы одно из полей пищевой ценности (калории, белки, жиры или углеводы).

#### Шаги

1. Зайти на сайт;
2. Открыть рецепт.

#### Ожидаемый результат 

В описании рецепта под блоком ингредиентов и главной фотографии отображается поле с КБЖУ.

#### Статус

Пройден


### Сценарий 2

Добавление КБЖУ к описанию рецепта при его создании.

#### Предусловие

- Пользователь авторизован.

#### Шаги

1. Зайти на сайт;
2. Открыть форму создания рецепта;
3. Ввести данные в блок КБЖУ.

#### Ожидаемый результат 

Пользователь беспроблемно заполняет поля формы, связанные с КБЖУ. Итоговый рецепт содержит блок КБЖУ.

#### Статус

Пройден

#### Предусловие

#### Шаги

2. Открыть форму редактирования рецепта;
3. Ввести данные в блок КБЖУ.

#### Ожидаемый результат 

Пользователь беспроблемно заполняет поля формы, связанные с КБЖУ. Итоговый рецепт содержит блок КБЖУ.

#### Статус

Пройден
