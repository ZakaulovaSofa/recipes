# Тестирование функциональности статей

## Тест-кейс №1

### Сценарий для теста

Просмотр детальной страницы статьи

### Предусловие

- Статья опубликована

### Шаги

1. Открыть страницу со списком всех статей
2. Открыть понравившуюся статью

### Ожидаемый результат

Откроется детальная страница статьи с фотографией и описанием статьи

### Статус

Пройден

## Тест-кейс №2

### Сценарий для теста

Комментирование статьи

### Предусловие

- Пользователь авторизован
- Статья опубликована

### Шаги

1. Открыть страницу со списком всех статей
2. Открыть нужную вам статью
3. Перейти в блок комментариев внизу страницы
4. Оставить свой собственный комментарий

### Ожидаемый результат

- Комментарий появится среди других комментариев на странице
- Есть возможность редактирования и удаления комментария для пользователя.

### Статус

Пройден

## Тест-кейс №3

### Сценарий для теста

Создание и публикация статьи

### Предусловие

- Пользователь имеет роль "Администратор"

### Шаги

1. Открыть страницу со списком всех статей
2. Нажать на кнопку "Создать статью"
3. Загрузить фото для статьи
4. Написать заголовок и описание статьи
5. Опубликовать статью

### Ожидаемый результат

Статья появится на странице списка всех статей

### Статус

Пройден

## Тест-кейс №4

### Сценарий для теста

Редактирование статьи

### Предусловие

- Пользователь имеет роль "Администратор"
- Статья опубликована

### Шаги

1. Открыть страницу со списком всех статей
2. Нажать на статью, которую нужно изменить
3. Нажать на кнопку "Редактировать статью", которая находится в самом низу детальной страницы статьи
4. Отредактировать нужный фрагмент(фото/заголовок/описание статьи) на странице редактирования
5. Сохранить изменения

### Ожидаемый результат

Статья будет опубликована со всеми внесёнными изменениями

### Статус

Пройден

## Тест-кейс №5

### Сценарий для теста

Удаление статьи

### Предусловие

- Пользователь имеет роль "Администратор"
- Статья опубликована

### Шаги

1. Открыть страницу со списком всех статей
2. Нажать на статью, которую нужно удалить
3. Долистать до самого низа детальной страницы и нажать на кнопку "Удалить статью"
4. В появившемся окне нажать на кнопку "Да"

### Ожидаемый результат

Статья будет удалена со страницы списка статей

### Статус

Пройден

## Тест-кейс №6

### Сценарий для теста

Попытка неавторизованного пользователя оставить комментарий

### Предусловие

- Пользователь не авторизован
- Статья опубликована

### Шаги

1. Открыть детальную страницу любой статьи
2. Пролистать вниз до блока комментариев

### Ожидаемый результат

- Форма ввода комментария не отображается
- Отображается текст: "Войдите, чтобы оставить комментарий" со ссылкой на страницу авторизации
- Комментарии, оставленные другими пользователями, видны

### Статус

Пройден

## Тест-кейс №7

### Сценарий для теста

Пагинация списка статей

### Предусловие

- Опубликовано более 20 статей

### Шаги

1. Открыть страницу со списком всех статей
2. Нажать на стрелку -> (вперед)
3. Нажать на стрелку <- (назад)

### Ожидаемый результат

- При нажатии -> отображается следующая страница со статьями, URL меняется
- При нажатии <- возвращается на первую страницу
- Стрелка <- отсутствует на первой странице

### Статус

Пройден 

## Тест-кейс №8

### Сценарий для теста

Валидация обязательных полей при создании/редактировании статьи

### Предусловие

- Пользователь имеет роль "Администратор"

### Шаги

1. Открыть страницу создания новой статьи
2. Не заполнять заголовок, нажать "Опубликовать"
3. Не заполнять описание, нажать "Опубликовать"
4. Не загружать фото, нажать "Опубликовать"

### Ожидаемый результат

- Форма не отправляется
- Показывается сообщение о необходимости заполнить поле
- Статья не создаётся

### Статус

Пройден

## Тест-кейс №9

### Сценарий для теста

Отсутствие кнопок управления для обычного пользователя

### Предусловие

Пользователь авторизован с ролью НЕ Администратор

### Шаги

1. Открыть страницу со списком всех статей
2. Открыть детальную страницу статьи

### Ожидаемый результат

- На странице списка всех статей нет кнопки «+ Добавить статью»
- На детальной странице статьи нет кнопок «Редактировать статью» и «Удалить статью»
- Блок комментариев для авторизованного пользователя присутствует

### Статус

Пройден
