# Тестирование функциональности статей

## Тест-кейс №1

### Сценарий для теста

Просмотр детальной страницы повара

### Предусловие
Карточка повара опубликована

### Шаги

1. Открыть страницу со списком всех поваров
2. Нажать на карточку или имя интересующего повара

### Ожидаемый результат

Откроется детальная страница повара c ФИО, фото, рейтингом(средней оценкой), категориями кухни (тегами), описанием/биографией

### Статус

Пройден

## Тест-кейс №2

### Сценарий для теста

Пагинация списка поваров

### Предусловие

- Опубликовано более 21 повара

### Шаги

1. Открыть страницу со списком всех поваров
2. Нажать на стрелку → (вперед)
3. Нажать на стрелку ← (назад)

### Ожидаемый результат

- При нажатии → отображается следующая страница с поварами
- При нажатии ← возвращается на первую страницу
- Стрелка ← отсутствует на первой странице

### Статус

Пройден 

## Тест-кейс №3

### Сценарий для теста

Создание и публикация повара

### Предусловие

- Пользователь имеет роль "Администратор"

### Шаги

1. Открыть страницу со списком всех поваров
2. Нажать на кнопку "Добавить статью"
3. Загрузить фото для повара
4. Написать ФИО повара, категории кухни и информацию о поваре.
5. Опубликовать повара

### Ожидаемый результат

Повар появится на странице списка всех поваров

### Статус

Пройден

## Тест-кейс №4

### Сценарий для теста

Валидация обязательных полей при создании/редактировании поваров

### Предусловие

- Пользователь имеет роль "Администратор"

### Шаги

1. Открыть страницу создания нового повара
2. Не загружать фото, нажать "Опубликовать"
3. Не заполнять фио, нажать "Опубликовать"
4. Не заполнять информацию, нажать "Опубликовать"

### Ожидаемый результат

- Форма не отправляется
- Показывается сообщение о необходимости заполнить поле
- Повар не создаётся

### Статус

Пройден

## Тест-кейс №5

### Сценарий для теста

Редактирование повара

### Предусловие

- Пользователь имеет роль "Администратор"
- Повар опубликован

### Шаги

1. Открыть страницу со списком всех поваров
2. Нажать на повара, которого нужно изменить
3. Нажать на кнопку "Редактировать повара", которая находится в самом низу детальной страницы повара
4. Отредактировать нужный фрагмент(фото/фио/теги/информацию) на странице редактирования
5. Сохранить изменения

### Ожидаемый результат

Повар будет опубликован со всеми внесёнными изменениями

### Статус

Пройден

## Тест-кейс №6

### Сценарий для теста

Удаление повара

### Предусловие

- Пользователь имеет роль "Администратор"
- Повар опубликован

### Шаги

1. Открыть страницу со списком всех поваров
2. Нажать на повара, которого нужно удалить
3. Долистать до самого низа детальной страницы и нажать на кнопку "Удалить повара"
4. В появившемся окне нажать на кнопку "Да"

### Ожидаемый результат

Повар будет удален со страницы списка поваров

### Статус

Пройден

## Тест-кейс №7

### Сценарий для теста

Отсутствие кнопок управления для обычного пользователя

### Предусловие

Пользователь авторизован с ролью НЕ Администратор

### Шаги

1. Открыть страницу со списком всех поваров
2. Открыть детальную страницу повара

### Ожидаемый результат

- На странице списка всех поваров нет кнопки «+ Добавить повара»
- На детальной странице повара нет кнопок «Редактировать повара» и «Удалить повара»

### Статус

Пройден

## Тест-кейс №8

### Сценарий для теста

Отметка оценки неавторизованным пользователем

### Предусловие

- Пользователь неавторизован
- Повар опубликован

### Шаги

1. Открыть страницу со списком всех поваров
2. Открыть нужного повара
3. Перейти к блоку рейтинга
4. Увидеть предупреждение о необходимости авторизации для оставления оценки

### Ожидаемый результат

- Не иметь возможности поставить оценик
- Под блоком рейтинга будет предупреждение: "Войдите, чтобы оценивать поваров."

### Статус

Пройден

## Тест-кейс №9

### Сценарий для теста

Отметка оценки авторизованным пользователем

### Предусловие

- Пользователь авторизован
- Повар опубликован

### Шаги

1. Открыть страницу со списком всех поваров
2. Открыть нужного повара
3. Перейти к блоку рейтинга
4. Выбрать оценку от 1 до 5 в выпадающем списке
5. Нажать кнопку "Поставить оценку"

### Ожидаемый результат

- В блоке рейтинга появится рейтинг(среднюю оценку) повара; количество оценок повара, увеличенное на 1
- Под блоком рейтинга появятся кнопки "Изменить оценку" и "Удалить".

### Статус

Пройден

## Тест-кейс №10

### Сценарий для теста

Изменение оценки авторизованного пользователя

### Предусловие

- Пользователь авторизован
- Повар опубликован
- Оценка проставлена

### Шаги

1. Открыть страницу со списком всех поваров
2. Открыть нужного повара
3. Перейти к блоку рейтинга
4. Выбрать новую оценку от 1 до 5 в выпадающем списке
5. Нажать кнопку "Изменить оценку"

### Ожидаемый результат

- Оценка смениться на новую
- Рейтинг пересчитается относительно новой оценки
- Количество оценок повара не изменится

### Статус

Пройден
