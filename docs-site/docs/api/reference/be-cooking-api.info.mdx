REST API для веб-платформы Be cooking — онлайн-кулинарной книги.

API предоставляет программный доступ к сущности рецепта:
получение списка рецептов, получение детальной информации,
создание, редактирование и удаление рецептов.

Основной пользовательский интерфейс приложения реализован через
HTML-страницы Flask/Jinja2, а данный API-слой используется для
обмена данными в формате JSON.
