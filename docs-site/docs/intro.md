# Be cooking

Be cooking — веб-платформа в формате онлайн-кулинарной книги для хранения, изучения и обмена кулинарными знаниями.

## Что реализовано

- рецепты пользователей;
- личный профиль;
- избранные рецепты;
- продуктовая корзина;
- комментарии;
- роли user/admin;
- REST API для рецептов;
- серверный интерфейс на Flask + Jinja2.

## Для кого проект

Проект ориентирован на пользователей, которые хотят хранить рецепты, планировать покупки и работать с личной кулинарной базой.
