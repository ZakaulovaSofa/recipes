# Статьи

## Что происходит с сущностью

Администратор может создавать, редактировать и удалять статьи, в то время как обычный пользователь может их просматривать.

## Где смотреть код

`/routes/articles_routes.py`

## Основные маршруты

- `GET /articles` | Список статей
- `GET /articles/<id>` | Детальный просмотр статьи
- `POST /admin/articles/add` | Создание статьи
- `POST /admin/articles/<id>/edit` | Изменение статьи
- `POST /admin/articles/<id>/delete` | Удаление статьи

## Ограничения доступа

Пользователь может просматривать статьи, администратор – добавлять, изменять и удалять.
