# Комментарии

## Что происходит с сущностью

Пользователь может создавать, просматривать, редактировать и удалять собственные комментарии, а также просматривать комментарии других пользователей..

## Где смотреть код

- `/routes/comment_routes.py`
- `/routes/recipe_routes.py`

## Основные маршруты

Получение списка комментариев осуществляется внутри рецепта / статьи при GET-запросе детального просмотра.

- `POST /recipes/<recipe_id>/comments` | Добавление комментария к рецепту
- `POST /comments/<comment_id>/edit` | Редактирование комментария
- `POST /comments/<comment_id>/delete` | Удаление комментария

## Ограничения доступа

Редактировать и удалять комментарий может только его автор.
