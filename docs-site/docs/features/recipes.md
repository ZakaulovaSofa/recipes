# Рецепты

## Что происходит с сущностью

Пользователь может создавать, просматривать, редактировать и удалять собственные рецепты.

## Где смотреть код

`/routes/recipe_routes.py`

## Основные маршруты

- `GET /recipes` | Список опубликованных рецептов 
- `GET /recipes/<id>` | Детальная страница рецепта
- `GET /my-recipes` | Получение списка рецептов пользователя
- `GET /recipes/<id>/manage` | Получение страницы рецепта с правом на редактирование
- `POST /recipes/add` | Создание рецепта 
- `POST /recipes/<id>/edit` | Редактирование рецепта 
- `POST /recipes/<id>/delete` | Удаление рецепта 

## Ограничения доступа

Редактировать и удалять рецепт может только его автор.
