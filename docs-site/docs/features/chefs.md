# Повара

## Что происходит с сущностью

Администратор может создавать, редактировать и удалять карточку повара, в то время как обычный пользователь может их просматривать.

## Где смотреть код

`/routes/chefs_routes.py`

## Основные маршруты

- `GET /chefs` | Список карточек поваров 
- `GET /chefs/<id>` | Детальная страница повара
- `POST /admin/chefs/add` | Создание карточки повара
- `POST /admin/chefs/<id>/edit` | Изменение карточки повара
- `POST /admin/chefs/<id>/delete` | Удаление карточки повара

## Ограничения доступа

Пользователь может просматривать карточку повара, администратор – добавлять, изменять и удалять.
