# Профиль

## Что происходит с сущностью

Пользователь видит информацию о своих недавно загруженных рецептах, избранном, а также информацию о своём профиле, включая свой уровень

## Где смотреть код

`/routes/user_routes.py`

## Основные маршруты

- `GET /profile` | Просмотр профиля
- `POST /profile/avatar` | Редактирование аватарки пользователя

## Ограничения доступа

Просматривать и редактированные данные профиля может только сам пользователь.
