# Продуктовая корзина

## Что происходит с сущностью

Пользователь может добавлять рецепты в корзину, удалять их оттуда и редактировать число порций.

## Где смотреть код

`/routes/cart_routes.py`

## Основные маршруты

- `GET /cart` | Страница продуктовой корзины
- `POST /cart/recipes/<id>/add` | Добавление рецепта в продуктовую корзину
- `POST /cart/recipes/<id>/servings` | Изменение числа порций рецепта
- `POST /cart/recipes/<id>/delete` | Удаление рецепта из корзины
- `POST /cart/clear` | Очищение продуктовой корзины

## Ограничения доступа

Только сам пользователь может менять состояние своей продуктовой корзины.
