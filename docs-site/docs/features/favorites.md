# Избранные рецепты

## Что происходит с сущностью

Пользователь может добавлять рецепты в избранное, удалять из него, а также просматривать избранное.

## Где смотреть код

`/routes/recipe_routes.py`

## Основные маршруты

- `GET /favorites` | Просмотр избранного
- `POST /recipes/<id>/favorite` | Добавление / удаление из избранного
- `POST /recipes/<id>/favorite/remove` | Удаление рецепта из избранного через модальное окно подтверждения


## Ограничения доступа

Добавлять и удалять рецепт из избранного может только авторизованный пользователь
