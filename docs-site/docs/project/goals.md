#  Цель и назначение системы

Цель проекта — разработка интерактивного кулинарного портала, который позволяет пользователям:
- делиться собственными рецептами и кулинарным опытом,
- изучать рецепты и статьи других авторов,
- структурировать личную кулинарную информацию,
- формировать индивидуальное пространство для планирования приготовления блюд и покупок.


---

## Границы проекта

Система **не включает**:
- автоматическую доставку продуктов или интеграцию с магазинами,
- создание подборок за исключением избранного,
- генерацию рецептов на основе искусственного интеллекта,
- реализацию чатов вне рецепта.
