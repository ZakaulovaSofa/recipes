# Описание проекта

**Be cooking** — это веб-платформа в формате онлайн-кулинарной книги, предназначенная для хранения, изучения и обмена кулинарными знаниями.  

Сайт объединяет рецепты, авторские статьи, заметки пользователей и справочную информацию о кулинарии в едином цифровом пространстве.

Проект ориентирован на создание удобной среды, где пользователи могут не только находить рецепты, но и формировать собственную кулинарную базу знаний.
